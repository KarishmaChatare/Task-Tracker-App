# Task Tracker App

## Features
- Add a task
- View all tasks
- Mark task as done
- Delete task

## Backend
- Java Spring Boot
- MySQL

Run backend:
```bash
cd backend
mvn spring-boot:run
```

## Frontend
- React, HTML, CSS

Run frontend:
```bash
cd frontend
npm install
npm start
```

Make sure MySQL is running and database `taskdb` is created.
